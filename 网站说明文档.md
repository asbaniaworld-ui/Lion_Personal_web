# Lion 个人站 — 内容结构与协作记录
**内容交互逻辑：**

本网站首先是在godly上参考了已有的网站，通过Chatgpt进行风格交流（图片+复刻），让其提供网页雏形和风格语句交由cursor实现基础模板；每个板块的内容有生成的底板，根据底板针对每个内容进行交互设计思考，结合gemini确定交互逻辑和内容，整合文字给cursor；在这个过程中，会通过截图、录屏、文字描述、具体代码指示，明确具体的网页设计细节和问题，大致格式：具体网页+问题直接描述（截图/录屏）+文字问题解释（分点列出）；内容链接过程中会将其下载到同根目录，进行引用；后期部署，直接交代用Github部署，根据网站要求为我调整相对链接、文档准备（gitignore）、整理网页涉及内容；最后完成部署并进行测试。




---

本文档整理当前网页各板块承载的内容与交互逻辑，并记录在本项目上与 Cursor 助手的协作过程（截至文档生成时的对话内容）。技术实现以仓库内 `index.html`、`styles.css`、`script.js`、`photo-corridor-3d.js` 为准。

## 一、站点概览

| 项目 | 说明 |
|------|------|
| 形态 | 静态站点（非 Next.js）：单页横向滚动「故事轨」+ 全屏引导层 |
| 首页入口 | `index.html` |
| 动效与滚动 | GSAP（含 ScrollTrigger、Flip） |
| 3D 照片廊 | Three.js（`photo-corridor-3d.js`） |

## 五、部署与观测（Vercel）

在 `index.html` 末尾（`</body>` 前）包含：

1. **Speed Insights**：`window.si` 引导 + `/_vercel/speed-insights/script.js`
2. **Web Analytics**：`window.va` 引导 + `/_vercel/insights/script.js`

静态站点在 Vercel 上托管时由平台注入上述路径；本地直接打开文件时这些请求可能无效，属预期现象。

**若控制台上看不到数据**，需核对：已用正确账号登录 Vercel、项目部署成功、用线上域名真实访问并等待数分钟；直接打开带 **Login** 的仪表盘链接通常是因为未登录或账号/团队不匹配。

## 六、与 Cursor 助手的协作记录（对话摘要）

以下内容按时间顺序概括你在本仓库相关任务中的诉求与已执行结果，便于论文、报告或交接时使用。

1. **Vercel 部署与 Speed Insights**  
   你已将站点迁至 Vercel。仓库为纯静态站，非 Next.js。助手在 `index.html` 中按官方静态方式接入 Speed Insights（非 React 组件 `<SpeedInsights />`），并说明与 Next 文档的差异。

2. **GitHub 与 Vercel 联动、提交推送**  
   你要求由助手完成提交并推送。助手将上述改动提交并推送到 `main`（推送过程中曾出现一次连接超时后重试成功）。

3. **接入 Web Analytics**  
   你回复「请」表示同意增加访问统计。助手增加 `window.va` 与 `/_vercel/insights/script.js`，同样经历一次推送失败后重试成功。

4. **Speed Insights 页面「无法显示」**  
   你提供的链接在未登录时会落到 Vercel 登录页。助手协助排查方向：账号/团队是否一致、部署是否就绪、是否产生真实流量及数据延迟；代码侧接入已完成。

5. **开篇视频改为 GIF 并更新 GitHub**  
   你将环形区域的 MP4 换为可调灵的狮子挥手 GIF。助手修改 `index.html`（`<video>` → `<img src="...gif">`）、调整 `script.js` 中环上介质的 `play`/`pause` 仅在 `HTMLVideoElement` 时执行；仓库删除大体积 `*.mp4`，新增 `*.gif`，提交并推送。

6. **说明文档（本文）**  
   你要求整合各板块内容与交互记录，形成一份说明文档。即本文件。

7. **补充交互语句**  
   你要求把对各板块的**具体交互语句**写入说明文档。已将 `index.html` 可见文案、`script.js` 动态提示与无障碍标签整理为 **「三（附）各板块交互与引导语句汇编」**，含 Loader 全套阶段文案、Introduction 双区 hint、Hobbies 的 `data-full-note` 与英文独白全文、Guestbook 游戏规则、`Contact` 复制成功提示等。
